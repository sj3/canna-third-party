Odoo Making Tax Digital as a Service

Overview

This module enables Odoo Community and Enterprise to commit HMRC VAT return to HMRC api.

Purpose


Why we decide to provide the connector as a service?

API software credentials are issued by HMRC to the software vendor, so if something happens with those credentials all customers can be locked.
Now, since the code is visible on the App store, we could expose those credentials and that would lead to a uncontrolled situation. Also, doing this as a service will allow us to progressively improve other related MTD phases.

Features

  • We support both community and enterprise editions
  • Currently support versions 10.0, 11.0 and 12.0
  • Allows VAT Report submission through HMRC web services
  • Allows viewing VAT obligations and liabilities
  • Our module saves the relationship between each tax entry and the submitted report, this means that nothing stays behind.
  • We provide an easy form for calculating and journalizing the fuel scale charges on your Chart of Accounts

How to install

  • Firstly, ensure that the module file is present in the add-ons directory of the Odoo server
  • Update Modules list so that it appears in the UI within Apps store
  • Update Modules list so that it appears in the UI within Apps
  • Install the dependencies odoorpc and msgfy
  • Look for the module within Apps and click on Install

How to configure

  • Go to Accounting/Invoice -> Configuration -> Settings
    • Search for "HMRC MTD" and fill in the fields:
    • Login
    • Password
    Note: Those credentials are provided by us. Contact us to ask for your credentials.

How to use

  • Sign up for Making Tax Digital for VAT here.
  • Set up the MTD module on Odoo
  • Make the first VAT return
