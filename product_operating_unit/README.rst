======================
Product Operating Unit
======================

This module implements global security rules on products so that a user can
only access shared products or products linked to an operating unit that the user has access to.

Usage
=====

This module defines the operating unit entity and the user's security rules.
Other modules extend the standard Odoo apps with the OU.
