=====================================
Account Entries Report Operating Unit
=====================================

This module introduces the operating unit as a dimension for analysis for
the Entries Analysis Report.


Usage
=====

* Go to 'Reporting / Accounting / Entries Analysis'. The field 'Operating
  Unit' is now an additional dimension for analysis.
